# Browser_Navigation_System
## DSA Assignment 1

Web Browser Navigation System using Stack, Queue, List, and DeQueue

### Overview
This project implements a Web Browser Navigation System using fundamental Data Structures and Algorithms concepts in Java. The system simulates real-world browser navigation features such as visiting pages, going back, going forward, and viewing browsing history.
The project demonstrates the practical use of Abstract Data Types (ADTs) including Stack, Queue, List, and DeQueue.
This project was developed as part of the COMP47500 Advanced Data Structures assignment.
________________________________________
### Features
•	Visit new web pages
•	Navigate backward using Stack (LIFO)
•	Navigate forward using Stack (LIFO)
•	Store browsing history using Queue (FIFO)
•	Maintain full history using List
•	Maintain recent history using DeQueue
•	Performance experiment and analysis included
